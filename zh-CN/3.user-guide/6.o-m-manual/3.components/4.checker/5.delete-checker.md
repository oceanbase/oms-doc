# 删除 Checker

为减少冗余的 Checker，您可以删除已完成和异常状态的 Checker。

## 操作步骤

1. 登录 OMS 社区版控制台。

2. 在左侧导航栏，单击 **运维监控** **\>** **组件**。

3. 在 **组件** 页面，单击 **Checker** 页签。

4. 在 **Checker** 页面，您可以通过 Checker 名称、机器 IP、迁移项目的名称或 ID 进行查询，查找目标 Checker。

5. 单击目标 Checker 后的 **...** \> **删除**。

6. 在 **删除确认** 对话框中，输入 Checker 名称以确认操作。

7. 单击 **删除**。
