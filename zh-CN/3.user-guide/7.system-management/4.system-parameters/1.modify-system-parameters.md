# 查看和修改系统参数

OceanBase 迁移服务（OceanBase Migration Service，OMS）社区版支持 Admin 用户修改系统参数，普通用户仅支持查看系统参数。

## 操作步骤

1. 登录 OMS 社区版控制台。

2. 在左侧导航栏，单击 **系统管理** > **系统参数**。

   **系统参数** 页面为您详细列举每个配置项的 **参数名称**、**取值**、**所属模块**、**描述** 和 **修改时间**。

3. 单击目标参数 **取值** 列下的编辑图标。

4. 在 **修改取值** 对话框中，配置 **取值**，您也可以 **恢复默认值**。

5. 单击 **确定**。
