# Source Plugin 概述

Source Plugin 的配置路径为 `conf/source.json`，每个配置类都存在一个 config key。

## 【启动】boosterClass

* 功能：指定 Source Plugin 的 Booster

* 默认值：无，必填

* 值的选择范围：下述 Plugins 或者额外实现的 plugin.jar 的 Booster

* 支持版本：OMS 社区版 V4.1.1-CE 及之后版本
